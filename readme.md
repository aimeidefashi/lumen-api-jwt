### 使用 Lumen 搭建 RESTful Api
## Lumen PHP Framework

## 关于

- [JWT Auth](https://github.com/tymondesigns/jwt-auth) 
- 通过请求头携带: `Authentication` 认证

## 快速开始

- 克隆 或 下载这个项目
- 执行 `composer install`
- 创建并编辑配置文件 `.env` 
- 创建 `storage` 并赋予权限
- 执行 `php artisan migrate --seed`  


## 测试
* 可导入 `lumen-jwt-auth.postman_collection.json` 至 POSTMAIN 工具测试
